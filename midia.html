Imagens

Para utilizarmos imagems no documento html5 basta apenas usarmos a tag <img src="x" alt="y">

Podemos carregar imagens que estão na mesma pasta do index.html também podemos carregar imagens que estão em outra pasta e podemos carregar imagens usando links externos

Imagens Dinamicas

Podemos colocar imagens dinamicas em sites usando a tag <picture></picture> e dentro da tag picture podemos usamos a tag já conhecida <img> para colocarmos a imagem principal, e depois usamos a tag <source media="(min-width: )" srcset="" type="image/"> sempre da maior imagem para a maior, veja o exemplo: 

 
<picture>
        <source media="(max-width: x)" srcset="x" type="image/png"> Imagem Pequena
        <source media="(max-width: x)" srcset="x" type="image/png"> Imagem média
        <img src="x" alt="y"> Imagem Original maior
</picture>

Note que a tag <source> possui três autributos:

type vai indicar o media type da imagem

srcset vai configurar o nome da imagem que será carregada quando o tamanho indicado for atingido

media indica o tamanho maximo a ser considerado para carregar a imagem indicada no autributo srcset

Áudio

Podemos adicionar áudio aos documentos em html5 usando a tag <audio> e algumas tags <source>

Exemplo:

    <audio src="x"></audio>

Outra forma de adicionarmos é

    <audio preload="auto" autoplay controls> Tag principal com alguns atributos que serão explicados logo abaixo
        <source src="midia/guanacast-33.mp3" type="audio/mpeg"> Primeira opção que o navegador tentará carregar, caso não consiga irá para o source de baixo
        <source src="midia/guanacast-33.ogg" type="midia/ogg"> Podemos adicionar arquivos de diferentes tipos para diferentes navegadores tentarem carregar
        <source src="midia/guanacast-33.wav" type="midia/wav">  Arquivos .mp3 .ogg e .wav são suportados
        <p>Infelizmente seu navegador não suporta áudio, Baixe o arquivo de áudio <a href="midia/happy-mistake.mp3" dawload>aqui</a></p> 
    </audio> Caso nenhum source seja carregado, Apareceça essa mensagem ↑

Principais Atributos da tag <audio>

preload
indica se o áudio será pre-carregado ou não aceitando três valores:
metadata: carrega apenas informações sobre o arquivo.
none: não irá carregar nada, até que seja iniciado a reprodução
auto: irá carregar todo o arquivo de audio assim que a página for carregada
controls
Irá apresentar o player na tela, caso não seja colocado na tag, o usuário não poderá interagir com o player e ele não aparecerá
autoplay
vai iniciar a reprodução do áudio assim que a página for carregada
loop
força o audio a ser repetido eternamente assim que terminar sua reprodução
