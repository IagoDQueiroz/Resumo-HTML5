Tags e Atributos em CSS3

Formas de aplicar as CSS nos seus documentos HTML5.

Podemos aplicar as CSS de 3 formas diferentes:

CSS Inline

Podemos aplicar as CSS na mesma linha que a tag a ser estilizada, exemplo: 

    <h1 style="x">

CSS Interno

Podemos aplicar as CSS de forma interna usando a tag <style>entro da tag <head> 

<head>
    <style>
        h1{x}
        h2{x}             
     </style>
</head>

CSS Externo

Podemos aplicar as CSS de forma externa, usando outro arquivo e herdando suas CSS para o principal, usando a tag <link rel="stylesheet" href="style.css"> nesse outros arquivo utilizamos de Seletores e propriedades como no exemplo Interno
