Títulos e Parágrafos

Podemos criar Títulos usando a tag <hx> indo do nivel 1 de hierarquia até o nivel 6.

Podemos criar parágrafos utilizando a tag <p>

Para quebrar a frase no meio dela podemos utilizar a tag <br>

Para criar linhas horizontais podemos utilizar a tag <hr>
