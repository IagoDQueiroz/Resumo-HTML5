Litas em HTML5

Listas podem ser criadas em conjunto, Listas dentro de listas

Listas Ordenadas

<ol type="x" start="y"> Lista Ordenada começam com a tag <ol>
<li> Item 01 </li>   Itens são adicionados as listas usando a tag, não a limite de itens <li>
<li> Item 02 </li>
<li> Item 03 </li>
</ol> Fechamento Obrigatório

Listas não Ordenadas

<ul type="x"> Listas não Ordenadas começam com a tag <ul>
<li> Item 01 </li> Itens são adicionados as listas usando a tag, não a limite de itens <li>
<li> Item 02 </li>
<li> Item 03 </li>
<ul> Fechamento Obrigatório

Listas de Definição

<dl>  Toda lista de Definição começa com a tag <dl>
<dt></dt> Termo que vai ser definido
<dd><dd>  Descrição do termo definido
<dl>
Pode se repetir esse padrão até quando for nescessário
